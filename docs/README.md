# Kiroku Manga Reader

Kiroku Manga Reader — простий Android-застосунок для читання манґи, що зберігається
локально на пристрої. Застосунок не завантажує контент із мережі: користувач
обирає кореневу директорію, після чого застосунок показує знайдені манґи, глави
та сторінки.

## Можливості

- вибір директорії через Android Storage Access Framework;
- збереження дозволу на читання вибраної директорії;
- відновлення списку манґи після перезапуску без повторного сканування;
- підтримка манґи з окремими директоріями глав і без них;
- природне сортування назв (`chapter 2` перед `chapter 10`);
- читання сторінок горизонтальними свайпами;
- масштабування жестом і подвійним дотиком;
- приховування панелі читача дотиком;
- ручне перемикання портретної та альбомної орієнтації;
- асинхронне й паралельне сканування файлів.

## Вимоги до структури бібліотеки

Застосунок підтримує два варіанти організації файлів.

### Манґа з главами

```text
Manga Library/
├── Manga A/
│   ├── Chapter 1/
│   │   ├── 001.jpg
│   │   └── 002.jpg
│   └── Chapter 2/
│       ├── 001.webp
│       └── 002.webp
└── Manga B/
    └── Chapter 1/
        └── page.png
```

### Манґа без окремих глав

Якщо в директорії манґи немає вкладених директорій, усі зображення в ній
вважаються однією главою.

```text
Manga Library/
└── One Shot/
    ├── 001.jpg
    ├── 002.jpg
    └── 003.jpg
```

Підтримувані розширення: `png`, `jpg`, `jpeg`, `webp`, `gif`.

## Використання

1. Запустіть застосунок.
2. Натисніть плаваючу кнопку `+`.
3. Надайте застосунку доступ до файлів, якщо Android його запитає.
4. Виберіть кореневу директорію, яка містить директорії манґи.
5. Виберіть манґу, потім главу.
6. Гортайте сторінки свайпами. Використовуйте pinch-to-zoom або подвійний дотик
   для масштабування.

Після сканування список манґи зберігається локально та відновлюється під час
наступного запуску застосунку. Щоб вибрати іншу бібліотеку або оновити список
після зміни файлів, повторно натисніть кнопку `+` і виберіть кореневу директорію.

Під час масштабування свайпи між сторінками блокуються, щоб жест рухав поточне
зображення. Одинарний дотик по сторінці показує або приховує панель читача.

## Поточні обмеження

- Список манґи кешується після сканування, але зміни у файловій системі не
  з'являться до повторного вибору та сканування кореневої директорії.
- Підтримуються лише директорії та окремі файли зображень; архіви `CBZ`, `ZIP`
  та `RAR` не підтримуються.
- Немає збереження прогресу читання, закладок чи історії.
- Тексти інтерфейсу переважно задані безпосередньо в коді та XML і наразі не
  локалізовані.
- Автоматизовані тести відсутні: шаблонні тестові файли перейменовані в `.bak`.
